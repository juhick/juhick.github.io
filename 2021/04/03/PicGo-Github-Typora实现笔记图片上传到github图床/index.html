PicGo+Github+Typora实现笔记图片上传到github图床

GIthub新建仓库并获取Token

首先创建一个公有库，因为若是私有库的话在外部是访问不到图片的

点击用户里的设置

点击开发者选项

点击个人token

点击生成新的token

只需要选择repo选项即可

将token保存下来，因为只有这一次机会可以复制，不要错过！！！

设置PicGo

然后下载PicGo，选择适合自己的版本即可

安装完成后设置Github图床

然后就可以测试是否可以上传了，因为某些原因(你懂的)，直接上传可能会失败，可以配置代理，就不再赘述，自行摸索。

配置Typora

打开偏好设置
